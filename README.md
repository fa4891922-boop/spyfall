# 🕵️ Шпион (Spyfall)

Многопользовательская браузерная игра «Шпион» на Node.js + Express + Socket.IO.

## Как играть
1. Один игрок создаёт комнату и получает 4-значный код (или делится ссылкой).
2. Остальные присоединяются по коду/ссылке со своих устройств.
3. Ведущий выбирает набор локаций, длительность раунда, время на ход и нажимает «Начать игру».
4. Всем игрокам, кроме одного, выдаётся **локация** и **роль**. Одному игроку — роль **Шпиона** (он не знает локацию, но получает тонкую подсказку).
5. Игроки по очереди задают друг другу вопросы. Цель местных — вычислить шпиона; цель шпиона — угадать локацию и не выдать себя.
6. Когда время выходит (или ведущий завершает раунд) — игра показывает локацию, роли всех игроков и счёт.

## Возможности
- 🎭 **Наборы локаций**: Классический (55), Города, Фильмы, История, Профессии (`data/locations/`).
- 🧩 **Спец-роли** (5+ игроков): Детектив (видит локацию среди ложных вариантов) и Напарник шпиона.
- ⏱️ **Гибкий таймер**: длительность раунда и время на ход (15/30/45/60 сек) выбираются в лобби.
- 🔁 **Reconnect**: при разрыве соединения роль и состояние восстанавливаются (по `sessionId` в localStorage).
- 👁️ **Режим зрителя**: зритель видит локацию и шпиона, но не участвует.
- 💬 **Чат** с быстрыми фразами и 🎤 **голосовыми сообщениями** (Web Audio / MediaRecorder).
- 🏆 **Таблица лидеров** и подробные **итоги раунда** (кто кем был, кто как голосовал).
- 🟣 **Аватарки** игроков (цвет по имени + первая буква).
- 🔊 **Звуковые эффекты** (Web Audio API, без файлов) и микроанимации.
- 🌍 **Мультиязычность** (RU/EN, автоопределение по браузеру) — `public/lang/`.
- 🤖 **Боты** для тестирования и демо (кнопка «+ Бот» в лобби).
- 🛡️ **Защита от спама**: rate-limit на сокет-события и HTTP API.
- 🧹 **Автоочистка** пустых комнат, лимит активных комнат (50), graceful-передача роли ведущего.
- 📈 **Мониторинг**: `/healthz`, `/stats` (комнаты, игроки, uptime, всего игр/игроков, популярные локации).
- 🚀 **Опциональный Redis** для горизонтального масштабирования (активируется через `REDIS_URL`).

## Запуск локально
```bash
npm install
npm start
```
Откройте http://localhost:3000

## Деплой на Render
- **Build Command:** `npm install`
- **Start Command:** `npm start`
- Render автоматически передаёт порт через переменную окружения `PORT`.
- (Опционально) задайте `REDIS_URL` для масштабирования на несколько инстансов.

## API
- `GET /healthz` — проверка живучести.
- `GET /stats` — статистика и аналитика (JSON).
- `GET /api/locations` — список наборов локаций.
- `GET /api/locations/:set` — локации конкретного набора.

## Технологии
- Node.js + Express — HTTP-сервер и статика
- Socket.IO — реальное время (комнаты, роли, таймер, чат, голосовые)
- Чистый HTML/CSS/JS на фронтенде (без сборки)
- Опционально: redis + @socket.io/redis-adapter, express-rate-limit

## Структура
```
server.js                 — игровой сервер (Socket.IO)
locations.js              — реэкспорт классического набора (back-compat)
data/locations/*.json     — наборы локаций + манифест index.json
data/roles.json           — конфигурация спец-ролей
public/index.html         — интерфейс
public/app.js             — клиентская логика
public/styles.css         — стили
public/lang/{ru,en}.json  — переводы
```
