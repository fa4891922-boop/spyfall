# 🕵️ Шпион (Spyfall)

Многопользовательская браузерная игра «Шпион» на Node.js + Express + Socket.IO.

## Как играть
1. Один игрок создаёт комнату и получает 4-значный код.
2. Остальные присоединяются по коду со своих устройств.
3. Ведущий выбирает длительность раунда и нажимает «Начать игру».
4. Всем игрокам, кроме одного, выдаётся **локация** и **роль**. Одному игроку — роль **Шпиона** (он не знает локацию).
5. Игроки по очереди задают друг другу вопросы. Цель обычных игроков — вычислить шпиона; цель шпиона — угадать локацию и не выдать себя.
6. Когда время выходит (или ведущий завершает раунд) — игра показывает локацию и кто был шпионом.

## Запуск локально
```bash
npm install
npm start
```
Откройте http://localhost:3000

## Деплой на Render
- **Build Command:** `npm install`
- **Start Command:** `npm start`
- Render автоматически передаёт порт через переменную окружения `PORT`.

## Технологии
- Node.js + Express — HTTP-сервер и статика
- Socket.IO — реальное время (комнаты, роли, таймер)
- Чистый HTML/CSS/JS на фронтенде (без сборки)

## Структура
```
server.js        — сервер и игровая логика
locations.js     — список локаций и ролей
public/          — фронтенд (index.html, styles.css, app.js)
package.json     — зависимости и start-скрипт
```
