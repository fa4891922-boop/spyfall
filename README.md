# 🕵️ Шпион (Spyfall)

Многопользовательская браузерная игра «Шпион» на Node.js + Express + Socket.IO.

## Как играть
1. Один игрок создаёт комнату и получает 4-значный код.
2. Остальные присоединяются по коду со своих устройств.
3. Хост выбирает набор локаций и длительность раунда, нажимает «Начать раунд».
4. Всем игрокам, кроме одного, выдаётся **локация**. Одному — роль **Шпиона** (он не знает локацию, но видит подсказку).
5. Игроки общаются голосом (Discord) и задают друг другу вопросы. Цель местных — вычислить шпиона; цель шпиона — угадать локацию и не выдать себя.
6. Голосование против подозреваемого — единогласное. Шпион может в любой момент угадать локацию.
7. Когда время выходит или хост завершает раунд — игра показывает локацию, шпиона и таблицу лидеров.

## Возможности
- 🎭 **Наборы локаций**: Классический (55), Города, Фильмы, История, Профессии (`data/locations/`).
- ⏱️ **Гибкий таймер**: круговой SVG-таймер с анимацией «warning/danger», длительность 1–15 мин.
- 🔁 **Reconnect**: при разрыве соединения роль и состояние восстанавливаются (по `sessionId` в localStorage).
- 🗳️ **Голосование**: единогласное обвинение игрока с модальным оверлеем.
- 🎯 **Угадывание шпионом**: сетка локаций с поиском и подтверждением.
- 📝 **Список локаций**: можно вычёркивать (tap) для отслеживания.
- 🏆 **Таблица лидеров**: счёт, количество раз шпионом, победы.
- 🛡️ **Защита от спама**: rate-limit на сокет-события.
- 🧹 **Автоочистка** пустых комнат (10 мин idle), лимит 50 комнат.
- 📈 **Мониторинг**: `/healthz`, `/stats`.

## Запуск локально
```bash
npm install
npm start
```
Откройте http://localhost:3000

## Деплой на Render
- **Build Command:** `npm install`
- **Start Command:** `npm start`
- Render автоматически передаёт порт через `PORT`.

## API
- `GET /healthz` — проверка живучести.
- `GET /stats` — статистика и аналитика (JSON).
- `GET /api/locations` — список наборов локаций.

## Технологии
- Node.js + Express — HTTP-сервер и статика
- Socket.IO — реальное время (комнаты, роли, таймер, голосование)
- Чистый HTML/CSS/JS на фронтенде (без сборки)

## Структура
```
server.js                 — игровой сервер (Socket.IO)
data/locations/*.json     — наборы локаций + манифест index.json
public/index.html         — интерфейс
public/app.js             — клиентская логика
public/styles.css         — стили
```
